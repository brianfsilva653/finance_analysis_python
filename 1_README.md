# Sistema de Automação Financeira 📊

## 📖 A História do Projeto
Este projeto nasceu de uma necessidade real de organização durante minha transição de carreira. Atualmente, trabalho na área da saúde, mas sou estudante de Análise e Desenvolvimento de Sistemas e busco minha primeira oportunidade (estágio) na área de TI.

Para gerenciar essa fase de mudança, decidi automatizar o controle das minhas finanças. Ao trocar de emprego recentemente, percebi que precisava de uma ferramenta mais precisa do que as planilhas comuns para entender minha nova realidade financeira.

## 💡 O Problema e a Solução
O sistema foi desenvolvido para substituir três planilhas manuais, automatizando cálculos complexos como:

1. *Cálculo de Salário Líquido:* O código processa o salário bruto e desconta automaticamente impostos e benefícios, mostrando quanto realmente entra em conta.
2. *Gestão de FGTS:* Uma funcionalidade dedicada para acompanhar os depósitos e o saldo do Fundo de Garantia.
3. *Planejamento de Futuro e Juros Compostos:* Projeções patrimoniais para garantir que a transição de carreira ocorra com segurança financeira.

## 🛠️ Por que Python?
Como eu já vinha aprofundando meus estudos em Python, decidi aplicar esse conhecimento para transformar processos manuais em automação. O uso da linguagem neste projeto permite:

* *Visualização Estruturada:* Converter a lógica de programação em saídas de dados organizadas, permitindo visualizar todas as informações em formato de planilha diretamente pelo código.
* *Prática Técnica:* Aplicar conceitos de manipulação de dados e lógica que já estudo, garantindo cálculos precisos e padronizados.
* *Histórico Evolutivo:* Criar um registro organizado da minha evolução financeira e do meu crescimento como desenvolvedor.
